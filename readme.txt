This plugin displays the WOW Index | A Modern Day Tape Reader For Today's Electronic Markets from algofutures

== Description ==

The WOW Index can be thought of a modern day tool for the old school art of Tape Reading.

The WOW Index displays order flow momentum within the SP 500, Dow 30, STOXX 50 and Crude Oil electronic futures markets.

Viewers can monitor whether buy or sell programs have dominance at any given time in any given market.

Settings within the module allow you to surface charts that are either geared towards day trading or to swing trading. 

All information is in real-time.

== Installation ==

1. Upload the wowindex folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place [wowindex] in a page or post where you want the WOW Index chart to appear
